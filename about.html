About this site

This is a simple static website template. It's built with plain HTML and CSS — no JavaScript frameworks, no build pipeline, no dependencies.

Getting started

Edit index.html and style.css to customise this site. Add new pages by creating more .html files in the same folder.

How it runs

The platform serves these files with a simple HTTP server. Any changes you push to your repository will be reflected immediately.

Going further

When you outgrow a static site, consider upgrading to the Node.js / React or Python templates which support server-side logic and dynamic content.
